### рҹҺҜ Objetivos

**Objetivo General**

**Objetivos EspecГӯficos**
1. Implementar **arquitectura MVC** separando vistas, controladores y lГіgica de negocio.  
2. DiseГұar **interfaz responsive (mobile first)** utilizando **Sass** para estilos escalables.  
3. Crear una **base de datos relacional** para productos, categorГӯas, usuarios y pedidos.  
4. Desarrollar **panel de administraciГіn** con operaciones **CRUD** (productos, usuarios, categorГӯas).  
5. AГұadir **carrito de compras** y **registro/inicio de sesiГіn** de usuarios.  
6. Aplicar **buenas prГЎcticas de seguridad** (validaciГіn, roles, sanitizaciГіn de datos, etc.).
