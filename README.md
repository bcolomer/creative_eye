## рҹӣ пёҸ GuГӯa de EjecuciГіn Local

Si deseas clonar y ejecutar este proyecto en tu mГЎquina local para desarrollo o pruebas, sigue estos pasos.

### Prerrequisitos
AsegГәrate de tener instalado:
* **Node.js** (v20+) y NPM (v10)+.
* **PHP** (v8.3+) y Composer (v2.8+).
* **LARAVEL** (v12.3+).
* **MySQL** ejecutГЎndose.
* **Angular CLI** (v18+): `npm install -g @angular/cli`

### 1. ConfiguraciГіn del Backend (API)

```bash
# Entra en la carpeta del backend
cd backend

# Instala las dependencias de PHP
composer install

# Configura las variables de entorno
cp .env.example .env
# IMPORTANTE: Abre el archivo .env y configura tus credenciales de BBDD (DB_DATABASE, DB_USERNAME, etc.)

# Genera la clave de encriptaciГіn
php artisan key:generate

# Ejecuta las migraciones y los seeders (datos de prueba)
php artisan migrate --seed

# Levanta el servidor local
php artisan serve

```

### 2. ConfiguraciГіn del Frontend (Cliente)

```bash
# Abre una nueva terminal y entra en la carpeta del frontend
cd frontend

# Instala las librerГӯas de Node
npm install

# Inicia el servidor de desarrollo
ng serve
```
### рҹ”җ Acceso al Sistema
Una vez levantado el entorno, puedes registrarte como un nuevo usuario desde el formulario de registro (/registro) para probar el rol de Cliente.
